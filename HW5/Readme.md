1. 	Создайте ClassLibrary с типом ArrayWorker, который будет уметь принимать массив, инвертировать его и возвращать. Массив передаем сами явно в метод, а не через консоль.
2.	ArrayWorker из задания 1 должен также уметь принимать int[] source(массив челых чисел), int index и int newValue. Логика: вернуть массив, который будет содержать все значения из source + новое значение newValue, которое вставляется по индексу в массив. Следовательно, все старые элементы, индекс которых больше index, будут иметь новый индекс -> старый индекс + 1. Вернуть новый массив.
3.	Создайте ClassLibrary с типом Cone. Конструктор типа принимает 2 параметра: радиус (r) и высоту (h). Так же в типе есть два метода, которые высчитывают площадь поверхности основания и полную площадь.
